# NES GAME

## 按键设置

| 游戏手柄 | 电脑键盘按键 |
| -------- | ------------ |
| Up       | ↑            |
| Down     | ↓            |
| Left     | ←            |
| Right    | →            |
| B        | X            |
| A        | Z            |
| Select   | Space        |
| Start    | Enter        |
