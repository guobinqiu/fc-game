# FC Gamehub

## 按键设置

| 手柄 | 电脑键盘 |
| -------- | ------------ |
| Up       | ↑            |
| Down     | ↓            |
| Left     | ←            |
| Right    | →            |
| B        | X            |
| A        | Z            |
| Select   | Space        |
| Start    | Enter        |
